# ViT-GCN混合模型架构分析

## 总体思路

本模型旨在实现多模态抑郁症识别，融合视频、音频时序特征与面部关键点空间特征。其核心架构为三分支融合：

- **分支1：ViT（视频+音频时序特征）**
- **分支2：AnatomicalGCN（面部关键点空间特征）**
- **分支3：层级融合（ViT-GCN交互+深度Transformer融合）**

融合后通过自适应注意力池化，自动识别关键帧，最终进行二分类。

---

## 1. ViT分支（视频+音频时序特征）

- 视频特征（video_dim）和音频特征（audio_dim）分别通过Conv1d投影至128维。
- LayerNorm对齐后，进入**双向CrossAttentionFusion**（视频查询音频，音频查询视频，2层堆叠）。
- 两模态特征拼接为256维，送入PatchEmbedding（patch_size=15），再进入6层Transformer。
- 通过CLS token进行分类。

### 主要流程

1. Conv1d投影
2. LayerNorm
3. CrossAttentionFusion（双向，2层）
4. 拼接
5. PatchEmbedding
6. Transformer
7. CLS token分类

---

## 2. GCN分支（面部关键点空间特征）

- 6个面部区域（mouth, nose, leye, reye, ljaw, rjaw）分别通过2层GCNConv提取空间特征。
- 区域内自注意力（MultiheadAttention）建模关键点依赖。
- 区域间跨区域注意力（MultiheadAttention）增强上下文。
- 全局支路：69个点的全局坐标投影，补充跨区域信息。
- 区域特征与全局特征拼接，LayerNorm归一化。
- 加入位置编码，送入1层Transformer进行时序建模。
- Attention Pooling自动识别关键帧。

### 主要流程

1. 区域GCNConv
2. 区域内自注意力
3. 区域间跨区域注意力
4. 全局坐标投影
5. 特征拼接+LayerNorm+位置编码
6. 时序Transformer
7. Attention Pooling

---

## 3. 层级融合分支（ViT-GCN交互+深度融合）

- ViT分支输出与GCN分支输出通过**CrossAttentionFusion**（ViT查询GCN，单向，1层，dropout=0.3）进行交互。
- 融合特征送入1层TransformerEncoder进行深度语义整合。
- Attention Pooling（MultiheadAttention+learned projection）自动选择关键帧。
- 最终特征经LayerNorm、Linear、ReLU、Dropout，输出2类（抑郁/非抑郁）。

### 主要流程

1. ViT-GCN CrossAttentionFusion
2. TransformerEncoder深度融合
3. Attention Pooling
4. 分类器输出

---

## 4. 训练与推理流程

- K折交叉验证（10折），10%留作测试集。
- Mixup数据增强，Label Smoothing交叉熵损失。
- 余弦学习率调度，early stopping（patience=25，min_delta=0.002）。
- 最优模型自动加载，最终测试评估。

---

## 5. 超参数与结构表

| 模块              | 结构/参数                                                          |
| ----------------- | ------------------------------------------------------------------ |
| ViT分支           | Conv1d→LN→CrossAttention(2层)→PatchEmbed→Transformer(6层)          |
| GCN分支           | GCNConv(2层/区域)→区域内/跨区域Attention→全局投影→Transformer(1层) |
| 层级融合          | CrossAttentionFusion(1层)→TransformerEncoder(1层)                  |
| Attention Pooling | MultiheadAttention(4头)+learned projection                         |
| 分类器            | LN→Linear(256→128)→ReLU→Dropout→Linear(128→2)                      |
| 损失函数          | LabelSmoothingCrossEntropy(smoothing=0.1)                          |
| 优化器            | AdamW, lr=1e-4, weight_decay=1e-2                                  |
| 学习率调度        | CosineAnnealingLR                                                  |
| Early Stopping    | patience=25, min_delta=0.002                                       |

---

## 6. 总结

本模型通过ViT与GCN的深度融合，结合层级交互与自适应注意力池化，实现了多模态时空特征的高效整合与关键帧自动识别，适用于抑郁症识别等复杂行为分析任务。
